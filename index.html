DS One

私有云 · NAS 管理 · 影音 · 音乐 · 电台 · 电子书

核心功能

  • 📁 群晖 NAS 文件管理 / 套件 / Docker 管理
  • 🎬 视频播放：Video Station / Emby / Jellyfin
  • 🎵 音乐播放：Audio Station / Navidrome / 电台
  • 📚 电子书阅读（OPDS 支持）
  • 🚗 CarPlay / 多端适配 / 深色模式
